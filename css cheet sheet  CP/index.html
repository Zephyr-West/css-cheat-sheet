Text styling

selector function example
Font:
font-family used to determin what font the text will use font-family: "Times new Roman";
Font-size used to determin how big the text will appear on screen and is set in pixels font-size: 18px;
font-weight determines how bold the text will appear in the web browser font-weight: bold;
font-style determines if the text is italicised or not font-style: italic;
color the colour that text will appear as in the web browser color: red;
typography:
text-align determine the alignment of the text text-align: center;
text-indent indents the first line of text in a paragraph by a set amount text-indent: 2em;
line-height determines the space between line of text in a paragraph and is set in em line-height: 2em;
text-transform used to make all text either uppercase or lowercase text-transform: uppercase;
letter-spacing sets the spacing between letters in pixels letter-spacing: 3px;
word-spacing sets the spacing between words word-spacing: 5px;
text-decoration determins wheather the text is underlined or overlined text-decoration: underlined;

Borders and lists

selector function example
borders:
border sets the style of the whole border of an html element including colour and thinkness border: 2px solid black;
border-top sets the style of just the top side of the border
border-right sets the style of just the right side of the border
border-bottom sets the style of just the bottom side of the border
border-left sets the style of just the left side of the border
border-width determines just the width of the border
border-color sets just the colour of the border
border-radius determines how rounded the corners of an element are border-radius: 25%;
lists:
list-style-type determines the style of the bullets or if any are used list-style-type: none;
list-style-position determines how text wraps list-style-position: inside;
list-style-image allows an image to be used as a bullet list-style-image: url(bullet.pgn);
